Detailed Description

template<>
class math::ode::EulerExpl< Surface >

This is the template to use for ODE's that are a function of time and x parameter: dx/dt = F( x(t), t ). One example is dx/dt = 2x + cos(t)

Member Function Documentation

void math::ode::EulerExpl< Surface >::Solve ( )
inline

Uses the Huen-Euler butcher tableau for a 1st and 2nd order Runge Kutta and compares the two. Adapts the time step by 2x or 1/2x according to RK1 and RK2 difference comparison.
