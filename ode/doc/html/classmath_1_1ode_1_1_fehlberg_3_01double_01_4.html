Detailed Description

template<>
class math::ode::Fehlberg< double >

This is the template to use for ODE's that are a function of a time and x parameter: dx/dt = F( x(t), t ). One example is dx/dt = 2x + cos(t)

void math::ode::Fehlberg< double >::Solve ( )
inline

Uses the Fehlberg butcher tableau for a 4th and 5th order Runge Kutta and compares the two. Adapts the time step by 2x or 1/2x according to RK4 and RK5 difference comparison.
