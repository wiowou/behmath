Detailed Description

template<template< template< typename > class Storage, typename T > class Sol, template< typename > class Storage = SparseVect, typename T = double>
class math::solver::Rectangular< Sol, Storage, T >

Solves rectangular m x n systems where m > n. The solver is a template parameter. Solves the system [A^T][C]( [b] - [A][x] ) = [f] -> [A^T][C][A][x] = [A^T][C][b] - [f]
